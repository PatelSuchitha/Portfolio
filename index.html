Projects

Static & Dynamic Security Analysis of Android Apps

Tools: FlowDroid, Soot, JADX

Analyzed 10 Android APKs for potential data leaks using taint analysis. Evaluated timeout impacts and documented sources, sinks, and leaks for each app.

View Detailed Findings
  • Timeout extension detects more leaks in certain APKs (e.g., mynt.app.apk, com.hawaiianairlines.app.apk).
  • Some APKs show no leaks, indicating strong data protection (e.g., com.wooxhome.smart.apk).
  • Detected leaks include location data, API tokens, clipboard access, media metadata, and contact data.
  • Decompiled APKs using JADX and analyzed privacy compliance.

AI & Cybersecurity: Adversarial Attack Study

Paper: "Why do adversarial attacks transfer? Explaining transferability of evasion and poisoning attacks" – Demontis et al., USENIX Security 2019

This project imitated the study to explore why adversarial attacks transfer between models. Reproduced experiments on evasion and poisoning attacks to understand attack transferability and model vulnerabilities.

View Key Insights
